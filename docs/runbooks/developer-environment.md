# Developer Environment

Recommended local setup for the Phase 1 workspace:

1. Create a Python 3.11+ virtual environment.
2. Install runtime dependencies from `requirements.txt`.
3. Install development tooling from `requirements-dev.txt`.
4. Run the desktop agent from `sandbox-agent/src/forensics_sandbox_agent/main.py`.

Notes:

- the desktop shell is scaffolded, not feature-complete
- simulator packages are separate Python deliverables
- the repository is organized for future PyInstaller packaging
